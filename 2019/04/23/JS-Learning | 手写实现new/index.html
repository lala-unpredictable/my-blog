JS-Learning | 手写实现new

JavaScript使用new时，函数调用会变成“构造函数调用”，new会劫持所有普通函数并用构造对象的形式来调用它。

在调用 new 的过程中会发生下面四件事情：

  1. 新生成了一个对象
  2. 链接到原型
  3. 绑定 this
  4. 返回新对象

接下来，我们来实现一下new


function _new() {
    // 创建一个空的对象
    let obj = new Object()
    // 获得构造函数
    let Con = [].shift.call(arguments)
    // 链接到原型
    obj.__proto__ = Con.prototype
    // 绑定 this，执行构造函数
    let result = Con.apply(obj, arguments)
    // 确保 new 出来的是个对象
    return typeof result === 'object' ? result : obj
}
