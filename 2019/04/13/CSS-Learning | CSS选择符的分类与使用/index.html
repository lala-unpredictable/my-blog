CSS-Learning | CSS选择符的分类与使用

CSS选择符

基本选择符

基本选择符包括：类型选择符、后代选择符、ID选择符以及类选择符。

  • 类型选择符（元素选择符）：用于选择特定类型的元素，比如段落，只要写出需要添加样式的元素名即可。
p {
    color: black;
}
  • 后代选择符：用于选择某个或某组元素的后代。（写法：在两个选择符中间==添加空格==）以下例子中，只有作为块引用后代的段落元素会被选中，从而缩进。
blockquote p {
    padding-left: 2em;
}

  • ID选择符：通过对应==ID属性的值==来选择元素。（写法：由==井号 #== 开头）

  • 类选择符： 通过对应==class属性的值==来选择元素。（写法：由句点 . 开头）

#intro {
    font-weight: bold;
}
.date-posted {
    color: #ccc;
}
<p id="intro">Happy Birthday, Anna</p>
<p class="date-posted">2019/04/13</p>

高级选择符

高级选择符包括：子选择符、相邻同辈选择符、一般同辈选择符、通用选择符、属性选择符、伪元素、伪类、结构化伪类、表单伪类。

  • 子选择符： 只选择一个元素的直接后代，也就是子元素，==孙子元素不受影响==。(写法：使用 > 子组合子)
#nav > li {
    color: red;
}

<ul id="nav">
    <li>第一集</li> // 显示为红色
    <li>
        <ul>
            <li>第二集（第一部分）</li> // 不受影响
            <li>第二集（第二部分）</li> // 不受影响
        </ul>
    </li>
    <li>第三集</li> // 显示为红色
</ul>
  • 相邻同辈选择符： 选择位于某个元素后面，并与该元素相邻并拥有共同父元素的元素。（写法：使用 ==+ 相邻同辈组合子==）
h2 + p {
    font-size: 1.4em;
    font-weight: bold;
    color: #777
}
  • 一般同辈选择符： 选择位于某个元素后面，所有与该元素拥有共同父元素的元素。（写法： 使用 ==～ 一般同辈组合子==）
h2 ～ p {
    font-size: 1.4em;
    font-weight: bold;
    color: #777
}
  • 通用选择符： 可以匹配页面中的所有元素。可以与组合子结合使用。(写法： 使用一个==星号== * )
// 选择类名为product-section的元素的直接后代，不管是什么元素，或者有什么属性

.product-section > * {
    /* ... */
}
  • 属性选择符： 基于元素是否有某个属性或者属性是否有某个值来选择元素。
 // 给带有title属性的abbr元素添加不同的样式
abbr[title] {
    border-bottom: 1px dotted #999;
}

// 为所有type属性值为submit的input元素在鼠标指针悬停时显示手状光标
input[type="submit"] {
    cursor: pointer;
}

// 属性值匹配某个模式

// 匹配以某些字符开头的属性值，在等号前面加入插入符（^）
a[href^="http:"]

// 匹配以某些字符结尾的属性值，在等号前面加入美元符号（$）
img[src$=".jpg"]

// 匹配包含某些字符的属性值，在等号前面加入星号（*）
a[href*="/adbout/"]

// 匹配以空格分隔的字符串中的属性值，在等号前面加入波浪号（~）
a[rel~=next]

// 匹配开头是指定值或指定值后连着一个短划线的属性值，在等号前面加入竖线（|）
a[lang|=en]
  • 伪元素：不给页面添加额外的标记，使用伪元素来插入小图标及版面装饰符号。
// HTML
<h1>The World's Greatest Swordsman</h1>
<section class="chapter">
    <p>
        At an exhibition of the world's best swordsman, the third-place fencer took the stage. A fly was released, and with an arc of his sword he cut the fly in half. The crowd cheered. Then the second-place man sliced a fly into quarters. A hush fell in anticipation of the world's greatest swordsman.
    </p>
</section>

// CSS
.chapter::before { //.chapter开头装饰性大引号”
    content: '”';
    font-size: 15em;
}
.chapter p::first-letter { // 段落的首字下沉效果
    float: left;
    font-size: 3em;
}
.chapter p::first-line { // 第一行转换为全部大写
    text-transform: uppercase;
}
  • 伪类： 基于文档结构以外的情形来为页面添加样式。（写法：以==一个冒号==开头，用于选择元素的特定状态或关系）
    PS：（:target伪类匹配的元素有一个ID属性，而且该属性的值出现在当前页面URL末尾的井号后边。）
    // 为访问过的链接为蓝色
a.link {
 color: blue;
}
// 访问过的链接为绿色
a.visited {
 color: green;
}
// 链接在鼠标悬停及获取链接焦点时为红色
a.hover,
a.focus {
 color: red;
}
// 活动状态时为紫色
a.active {
 color: purple;
}
  • 结构化伪类：CSS3新增的 与==文档结构==有关的新伪类

// 常用的nth-child选择符，可以用来交替地为表格行应用样式
tr:nth-child(odd) { // 参数接受odd奇数和even偶数，也可以是数值或者数值表达式
    background: yellow;
}
// :nth-last-child选择符从最后一个元素倒序计算
// CSS3还增加了last-child与CSS2.1中的first-child对应
  • 表单伪类： 专门用于选择==表单元素==
// 使用:required伪类来选择带有required属性的表单元素
input:required {
    outline: 2px solid #000;
}
// 使用:optional伪类，为没有required属性的控件添加样式
input:optional {
    border-color: #ccc;
}
//...

以上为CSS选择符的分类及举例使用方法。
