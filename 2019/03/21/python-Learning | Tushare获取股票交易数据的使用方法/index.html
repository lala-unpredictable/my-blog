Python-Learning | Tushare获取股票交易数据的使用方法

Tushare

Tushare是一个免费、开源的python财经数据接口包。主要实现对股票等金融数据从数据采集、清洗加工到数据存储的过程，能够为金融分析人员提供快速、整洁和多样的便于分析的数据，为他们在数据获取方面极大地减轻工作量，使他们更加专注于策略和模型的研究与实现上。tushare拥有以下==特性==：

  • 数据丰富：
    tushare拥有丰富的数据内容，如股票、基金、期货、数字货币等行情数据，公司财务、基金经理等基本面数据
  • 获取简单：
    SDK开发包支持语言，同时提供HTTP Restful接口，最大程度方便不同人群的使用
  • 落地方便：
    提供多种数据储存方式，如Oracle、MySQL，MongoDB、HDF5、CSV等，为数据获取提供了性能保证

获取Tushare数据

准备工作

  1. 安装python（注意tushare不支持python3.7版本）
  2. 安装pandas，我是直接安装的Anaconda，包含有pandas。建议安装Anaconda，更方便。
  3. 安装tushare模块，可直接输入命令pip install tushare

获取数据

这里可以查看tushare官网（数据接口），各种数据接口都可以查看接口名和输入输出参数以及样例。
这里需要注意的是，获取数据需要token验证，要先注册并登陆后，在==个人主页》接口TOKEN》复制==后使用该token。
有需要的朋友可以点击该链接进行注册https://tushare.pro/register?reg=247661，成功注册一个用户即可获得50积分，有的数据是有积分需求的。

举个例子：获取沪深股票日线行情，接口名：daily

import tushare as ts
ts.set_token('你的token数据')
pro = ts.pro_api()
df = pro.daily(ts_code='000001.SZ', start_date='20180701', end_date='20180718')
df.to_excel('.../.../1.xlsx') // 储存数据在本地的某个excel文件中
